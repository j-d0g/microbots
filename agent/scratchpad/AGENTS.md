# scratchpad/

Numbered task folders for Jordan's harness notes. Agents must consult this file before task work, pick the active ticket, then follow that folder's local README or handoff.

These folders are planning/notes scratchpads, not implementation roots. Do not create runnable app code, virtualenvs, generated scaffolds, `.env` files, or deployment projects inside `agent/scratchpad/pN-*` unless Jordan explicitly asks for code to live there. Use them for notes, specs, handoffs, research, and test plans.

When a task needs real code, first confirm the intended implementation location with Jordan.

Current harness implementation code belongs under `agent/harness/` unless Jordan says otherwise.
